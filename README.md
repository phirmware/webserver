### Web Server Built With Go lang


- ##### Clone the application into your $GOPATH/src/
```go
git clone https://github.com/phirmware/webserver.git
```
- ##### To run postgres db
```go
cd /webserver/postgres/

./setupDB.sh
```

- ##### Run the application
```go
go run main.go
```

- ##### Go to port 8080 on your localhost
